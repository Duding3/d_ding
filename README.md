# 🎮 JiHye Mini-Game Hub

재미있는 미니게임을 즐길 수 있는 웹 게임 허브입니다.

## 🕹️ 게임 목록

### 테트리스 (Tetris)
- 클래식 테트리스 게임
- 모바일 & PC 지원
- 고스트 피스, 다음 블록 미리보기, 하드 드롭 기능
- 키보드 조작: `←→` 이동 | `↑` 회전 | `↓` 가속 | `Space` 하드드롭

### 타자 연습 (Typing)
- 한글/영어 타자 연습
- 실시간 정확도 및 타수(CPM) 측정
- 다양한 카테고리: 짧은 글, 긴 글, 동화, 영타연습
- 가상 키보드 시각화

## 🚀 실행 방법

1. `index.html` 파일을 브라우저에서 열기
2. 원하는 게임 선택 후 플레이!
